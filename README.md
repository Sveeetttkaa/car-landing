# Car Landing

Адаптивный лендинг на тему автомобилей.

## Demo

Живая версия сайта: https://landing-about-car.netlify.app/

## Функционал
- Адаптивный лендинг про электрические автомобили Porsche.
- Верхнее меню с бургер-иконкой для мобильной версии.
- Слайдер с карточками автомобилей и индикаторами-точками.
- Фильтры по брендам (все, Tesla, Audi, Lamborghini) для отображения нужных машин.

## Назначение проекта
Учебный/портфолио-проект для отработки адаптивной верстки и подготовки к работе Junior Frontend/верстальщиком.
